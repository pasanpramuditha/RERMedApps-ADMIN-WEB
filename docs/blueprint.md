# **App Name**: RER MedApps Dashboard

## Core Features:

- Authentication: Login page with username (admin) & password (admin) fields; redirects to /dashboard upon successful login.
- Dashboard Overview: Dashboard displays top stats cards (App Installs, Revenue, Subscriptions, etc.) with total value, trend indicator, platform breakdown, and mini chart.
- Total Income Section: Donut chart showing Lifetime, Monthly, and Yearly income with highlighted offer/discount data.
- Earning Breakdown Chart: Bar chart displaying earning breakdown per month (Jan-Dec) segmented by platform (Apple, Android, Admob).
- Sidebar Navigation: Persistent sidebar navigation menu with links to Dashboard, App Purchases, User Feedbacks, User Reviews, etc.
- App Purchases Page: Table view displaying user data (Name, Email, Join Date, Dept, Role, Status) with pagination, search filter, and actions (View, Edit, Delete). Includes '+ Add New User' button with modal form.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a professional and trustworthy feel.
- Background color: Light gray (#F0F0F5), offering a clean and neutral backdrop.
- Accent color: Soft orange (#FFAB40), used sparingly to highlight key interactive elements.
- Body and headline font: 'Inter' sans-serif for a clean, modern, and highly readable interface. This font is suitable for both headlines and body text.
- Use consistent, minimalist icons for navigation and actions to enhance usability.
- Responsive layout adapting to different screen sizes to ensure a consistent user experience.
